## 代理列表

| 文件名 | 代理名称 | 主要职责 |
|--------|----------|----------|
| agentic-identity-trust.md | 身份信任代理 | 代理身份验证、信任框架、安全编排 |
| report-distribution-agent.md | 报告分发代理 | 自动报告交付、定时分发 |

## 核心功能

| 功能 | 描述 |
|------|------|
| **代理编排** | NEXUS管道管理、阶段推进、质量门控制 |
| **数据管道** | Excel监控、指标提取、数据合并、报告分发 |
| **代码智能** | LSP实现、代码索引、符号导航 |
| **商业智能** | 深度分析、仪表板、KPI追踪 |

## 数据流程

```
Excel文件 → 销售数据提取代理 → 数据合并代理 → 报告分发代理
                ↓
        PostgreSQL数据库
                ↓
        数据分析报告员 → 商业洞察
```
