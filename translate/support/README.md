# 支持部门 (Support)

支持部门代理专注于客户服务、运营维护、财务跟踪和合规管理。

## 代理列表

| 文件名 | 代理名称 | 主要职责 |
|--------|----------|----------|
| support-support-responder.md | 支持响应者 | 多渠道客户服务、问题解决、用户体验 |
| support-infrastructure-maintainer.md | 基础设施维护者 | 系统可靠性、性能优化、云基础设施 |
| support-executive-summary-generator.md | 执行摘要生成者 | C级沟通、SCQA框架、战略报告 |
| support-analytics-reporter.md | 分析报告员 | 数据分析、仪表板、KPI追踪 |
| support-legal-compliance-checker.md | 法律合规检查者 | 合规、法规、法律审查、安全验证 |
| support-finance-tracker.md | 财务追踪者 | 财务规划、预算管理、ROI预测 |

## 支持渠道

| 渠道 | SLA响应时间 |
|------|-------------|
| 电子邮件 | 2小时 |
| 在线聊天 | 30秒 |
| 电话 | 3声振铃 |
| 社交媒体 | 1小时 |
| 应用内消息 | 即时(情境感知) |

## 关键指标

| 指标 | 目标 |
|------|------|
| 首次响应时间 | < 2小时 |
| 首次联系解决率 | 85% |
| 客户满意度 | > 4.5/5 |
| 系统正常运行时间 | > 99.9% |
| 合规遵守 | 98%+ |

## 服务层级

- **Tier 1**: 账户管理、基础故障排除、产品信息
- **Tier 2**: 高级故障排除、集成支持、定制配置
- **Tier 3**: 企业支持、定制开发、安全事件
