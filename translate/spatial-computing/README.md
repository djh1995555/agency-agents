# 空间计算部门 (Spatial Computing)

空间计算部门代理专注于沉浸式体验、Apple Vision Pro开发、XR界面设计和Metal渲染。

## 代理列表

| 文件名 | 代理名称 | 主要职责 |
|--------|----------|----------|
| macos-spatial-metal-engineer.md | macOS空间/Metal工程师 | Swift、Metal、高性能3D渲染、空间计算 |
| terminal-integration-specialist.md | 终端集成专家 | CLI工具、终端工作流、开发者工具集成 |
| xr-immersive-developer.md | XR沉浸式开发者 | WebXR、基于浏览器的AR/VR体验 |
| xr-interface-architect.md | XR界面架构师 | 空间交互设计、AR/VR/XR界面 |
| visionos-spatial-engineer.md | visionOS空间工程师 | Apple Vision Pro开发、空间应用 |
| xr-cockpit-interaction-specialist.md | XR座舱交互专家 | 基于座舱的控制、沉浸式控制界面 |

## 技术栈

| 平台 | 技术 |
|------|------|
| Apple Vision Pro | Swift、Metal、Compositor Services、visionOS |
| WebXR | WebXR API、Three.js、基于浏览器的AR/VR |
| macOS | Metal渲染、Swift、RemoteImmersiveSpace |
| 终端集成 | CLI工具、LSP、代码智能 |

## 性能要求

| 指标 | 目标 |
|------|------|
| 立体渲染帧率 | 90fps @ 25k节点 |
| GPU利用率 | < 80% |
| 注视到选择延迟 | < 50ms |

- 3D数据可视化
- 沉浸式代码编辑
