# 项目管理部门 (Project Management)

项目管理部门代理专注于项目规划、任务分解、跨职能协调和资源管理。

## 代理列表

| 文件名 | 代理名称 | 主要职责 |
|--------|----------|----------|
| project-management-project-shepherd.md | 项目牧羊人 | 跨职能协调、利益相关者对齐、项目执行 |
| project-manager-senior.md | 高级项目经理 | 规范到任务转换、现实范围设定、任务规划 |
| project-management-experiment-tracker.md | 实验追踪者 | A/B测试、假设验证、实验管理 |
| project-management-studio-operations.md | 工作室运营 | 日常效率、流程优化、协作工具 |
| project-management-studio-producer.md | 工作室制作人 | 战略组合对齐、执行编排、组合管理 |

## 项目管理框架

| 阶段 | 活动 |
|------|------|
| 规范分析 | 阅读规范、识别需求、提取技术栈 |
| 任务分解 | 将规范分解为30-60分钟可执行任务 |
| 冲刺规划 | 创建优先级待办事项、RICE评分 |
| 进度跟踪 | 监控任务状态、识别阻塞、资源分配 |
| 风险管理 | 范围蔓延控制、预算跟踪、时间线管理 |

## 交付物标准

| 交付物 | 内容 |
|--------|------|
| 任务列表 | 结构化开发任务，含验收标准 |
| 冲刺计划 | 优先级待办事项，含依赖图 |
| 架构规范 | 技术架构、CSS设计系统 |
| 预算框架 | 资源分配、成本预测 |

## 质量标准

- 任务验收标准清晰可测试
- 无范围蔓延
- 技术需求完整准确
- 时间线估算现实可行
