# 产品部门 (Product)

产品部门代理专注于市场研究、趋势分析、用户反馈处理和功能优先级排序。

## 代理列表

| 文件名 | 代理名称 | 主要职责 |
|--------|----------|----------|
| product-trend-researcher.md | 趋势研究员 | 市场情报分析、竞争分析、机会评估，提供可执行洞察 |
| product-feedback-synthesizer.md | 反馈综合者 | 多渠道反馈收集、情感分析、痛点优先级排序 |
| product-sprint-prioritizer.md | 冲刺优先者 | RICE评分、敏捷规划、功能优先级排序 |

## 核心能力

- **市场研究**：行业分析、竞争情报、市场规模估算
- **趋势分析**：模式识别、信号检测、未来预测
- **用户反馈**：多渠道反馈收集、痛点评分、需求验证
- **优先级排序**：基于价值的冲刺规划、待办事项管理

## 使用场景

- 产品开发前的市场机会评估
- 竞争格局分析和定位策略
- 产品路线图规划
- 功能优先级的用户研究验证
