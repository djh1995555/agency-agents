# 🚀 第5阶段剧本 — 发布与增长

> **持续时间**：2-4周（T-7到T+14） | **代理**：12个 | **门控者**：工作室制作人 + 分析报告器

## 目标

协调所有渠道同时执行市场推广。发布时最大化影响。每个营销代理同时发力，同时工程确保稳定性。

- [ ] 第4阶段质量门已通过（现实检查器就绪裁决）

### T-0：发布日

#### 第0小时：部署

```
激活DevOps自动化器：
1. 执行蓝绿部署到生产
2. 运行所有服务健康检查
3. 确认数据库迁移完成
4. 确认所有端点响应
5. 切换流量到新部署
6. 监控错误率15分钟
7. 确认：部署成功或回滚

激活基础设施维护者：
1. 实时监控所有系统指标
2. 观察流量峰值和扩展事件
3. 跟踪错误率和响应时间
4. 警报任何阈值违规
5. 确认：系统稳定
```

## 门决策

**双重签署**：工作室制作人（战略）+ 分析报告器（数据）

- **稳定**：产品已发布、系统稳定、增长活跃 → 第6阶段激活
- **关键**：需要立即工程响应的重大问题 → 热修复循环
- **回滚**：根本问题 → 回退部署，返回第4阶段
