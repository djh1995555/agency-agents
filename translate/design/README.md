# 设计部门 (Design)

设计部门代理专注于视觉设计系统、用户体验研究、品牌一致性和沉浸式界面创建。

## 代理列表

| 文件名 | 代理名称 | 主要职责 |
|--------|----------|----------|
| design-ui-designer.md | UI设计师 | 视觉设计系统、组件库、像素级完美界面 |
| design-ux-researcher.md | UX研究员 | 用户测试、行为分析、人物画像开发 |
| design-whimsy-injector.md | 趣味注入者 | 微交互、愉悦感、个性设计 |
| design-brand-guardian.md | 品牌守护者 | 品牌身份系统、一致性审计、定位策略 |
| design-image-prompt-engineer.md | 图像提示工程师 | AI图像生成提示、摄影艺术指导 |
| design-ux-architect.md | UX架构师 | CSS设计系统、布局框架、技术UX |
| design-visual-storyteller.md | 视觉叙事者 | 视觉叙事、多媒体内容、动画 |

## 设计系统要素

| 要素 | 内容 |
|------|------|
| 颜色系统 | 主色/辅色调色板、语义颜色、中性色 |
| 排版系统 | 字体家族、字体规模、字重、行高 |
| 间距系统 | 8点基础单位、一致间距比例 |
| 组件库 | 按钮、表单、导航、反馈组件 |
| 响应式断点 | 320px、640px、1024px、1280px |

## 可访问性标准

- WCAG AA合规（4.5:1对比度）
- 键盘导航支持
- 屏幕阅读器兼容性
- 触摸目标最小44px
