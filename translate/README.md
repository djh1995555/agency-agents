## NEXUS框架概览

本代理库基于 **NEXUS** (Network of Experts, Strategically Unified) 框架构建，定义了从发现到运营的完整产品生命周期。

### 七阶段管道

```
阶段0: 发现 → 阶段1: 策略 → 阶段2: 基础 → 阶段3: 构建 → 阶段4: 加固 → 阶段5: 发布 → 阶段6: 运营
```

### 部署模式

| 模式 | 活跃代理 | 用例 | 时间线 |
|------|----------|------|--------|
| NEXUS-Full | 全部 | 企业产品发布 | 12-24周 |
| NEXUS-Sprint | 15-25 | 功能/MVP | 2-6周 |
| NEXUS-Micro | 5-10 | Bug修复/内容活动 | 1-5天 |
